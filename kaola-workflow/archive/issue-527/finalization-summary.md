# Finalization - Summary: issue-527

## Delivered
Fix for bug #527: the gitlab and gitea `adaptive-node` `orient` `--summary` smoke tests (the
#446 anchor) ran `orient --project nonexistent-{gl,gt}-445-test --json --summary` from the
repo cwd. `orient`, though documented read-only, materializes
`kaola-workflow/<project>/.cache/orient-envelope.json` even for a nonexistent project, so each
`test:kaola-workflow:gitlab` / `test:kaola-workflow:gitea` chain run leaked an untracked
`kaola-workflow/nonexistent-{gl,gt}-445-test/` scratch project dir into the working tree that
the chain never cleaned up. Fix (test-local, narrowest blast radius): each test now runs its
`orient --summary` spawn in a `$TMPDIR`-rooted cwd (via the file's own `tempRoot(...)` helper)
and removes it in a `finally`, so the scratch lands in tmp and is cleaned even on assertion
failure. The shared `orient` source is untouched (direction (c) rejected for cross-edition
blast radius). The `--summary` sentinel assertion (`summaryOut.startsWith('summary:')`, #446)
is preserved unchanged. Applied as mirror twins to the gitlab + gitea test ports.
